Comment ajouter de l'interactivité à une présentation Reveal.js

C'est comme une page HTML5

Il suffit d'identifier les éléments de la présentation qu'on veut rendre interactifs et ajouter du code JavaScript. 

document
  .getElementById("btn-interactif")
  .addEventListener("click", function () {
    //Ajouter du texte à la slide
    document.getElementById("target-element").innerHTML = "Hello World !";
  });
